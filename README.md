# CodeAlpha Student Grade Tracker

## Overview

Student Grade Tracker is a Java console-based application that helps manage student records and grades. It allows users to add student details, view all records, calculate average scores, identify the highest and lowest scores, and search for students by name.

## Features

* Add student name and grade
* View all student records
* Calculate average score
* Find highest score
* Find lowest score
* Search student by name
* Console-based menu-driven interface

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Scanner

## How to Run

1. Clone the repository.
2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or VS Code).
3. Compile and run `StudentGradeTracker.java`.
4. Follow the menu displayed in the console.
